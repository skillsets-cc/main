# 404.astro

## Purpose
Static 404 error page displayed when a skillset or page is not found. Simple centered layout with large "404" heading and link back to browse page.

## Public API
| Export | Type | Description |
|--------|------|-------------|
| Page component | Astro page | 404 error page (static) |

## Dependencies
- **Internal**: `@layouts/BaseLayout.astro`
- **External**: None

## Integration Points
- **Used by**: Astro routing (displayed for non-existent URLs)
- **Consumes**: None
- **Emits**: No events

## Key Logic
- Centered text layout with vertical padding
- Large serif "404" heading
- Explanation message
- Call-to-action button linking to `/` (homepage)
